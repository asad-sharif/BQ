# Project Guide
Recreate the "Tech Book Club" design using HTML and CSS. Focus on clean structure and matching the layout.

- **Header:** Create a top navigation bar with the logo and a "Membership" button.
- **Hero Section:** Use a two-column layout to place the title text next to the main image.
- **Feature Blocks:** Build sections that alternate between text on the left and images on the right.
- **Step-by-Step Guide:** Create the "Reading Journey" section with numbered boxes and connecting lines.
- **Pricing Table:** Build three side-by-side cards for "Starter," "Pro," and "Enterprise."
- **Styling:** Use Flexbox for alignment and the Box Model to match the spacing and padding.
- **Footer:** Create a dark-colored section at the bottom for final links and icons.
