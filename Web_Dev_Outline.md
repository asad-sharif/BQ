# Web Development Fundamentals - Complete 12-Week Course Outline

**Course Name:** Web Development Fundamentals  
**Duration:** 12 Weeks | 24 Sessions (2 hours each)  
**Prerequisites:** None  
**Tools:** VS Code, Git, GitHub, Chrome DevTools, Bootstrap, Netlify account, Vercel account, JSONPlaceholder, Postman

---

## PHASE 1: HTML/CSS & FOUNDATIONS (Weeks 1-3)

| Week | Session | Focus Area | Topic & Key Concepts | Practical Project Outcome | Practice Tasks (Sunday) |
|------|---------|------------|----------------------|---------------------------|------------------------|
| **W1** | **S1** | HTML/CSS | **HTML5 & CSS Fundamentals** • Semantic HTML (header, main, section, article, footer) • CSS Box Model (margin, padding, border) • Colors, fonts, backgrounds • Class vs ID selectors • External CSS files • **Ref:** [MDN HTML Basics](https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/HTML_basics) • [MDN CSS Basics](https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/CSS_basics) | **Build:** Personal Profile Card - a self-contained card with profile picture, name, bio, and social links using semantic HTML and styled CSS | **Objective:** Build a complete 3-card section for a fictional company's team members. | **W1** | **S2** | CSS | **Responsive Design** • Flexbox (justify-content, align-items, flex-direction) • CSS Grid basics • Media queries • Mobile-first approach • **Ref:** [MDN Flexbox](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox) • [MDN Grid](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Grids) | **Build:** Make the Profile Card responsive - cards stack vertically on mobile, display in a row on desktop. | **W2** | **S3** | Tools | **Version Control & AI Tools** • Git workflow (init, add, commit, push, pull) • GitHub Pages deployment • Prompt Engineering basics (how to ask AI for code help) • Bootstrap introduction (grid system, components) • **Ref:** [GitHub Pages Docs](https://pages.github.com/) • [Bootstrap Docs](https://getbootstrap.com/docs/5.3/getting-started/introduction/) | **Build:** Deploy Profile Card to GitHub Pages. Use AI to generate a contact form and integrate Bootstrap for styling. | **Objective:** Create a new GitHub repo. Build a simple business landing page using Bootstrap components (navbar, cards, form). Deploy to GitHub Pages. |
| **W2** | **S4** | Framework | **Bootstrap Deep Dive** • Bootstrap grid system (container, row, col) • Bootstrap components (navbar, cards, modals, forms) • Customizing Bootstrap with CSS • **Ref:** [Bootstrap Grid](https://getbootstrap.com/docs/5.3/layout/grid/) • [Bootstrap Components](https://getbootstrap.com/docs/5.3/components/) | **Build:** Complete Business Landing Page - responsive hero section, services grid, testimonial cards, contact form with modal popup. | **Objective:** Build a portfolio website for a fictional photographer using Bootstrap. Include gallery grid, about section, and contact form. Deploy to GitHub Pages. |
| **W3** | **S5** | JavaScript | **Programming Fundamentals** • Variables (let, const) • Data types (string, number, boolean) • Operators (arithmetic, comparison) • Template literals • Conditionals (if/else, switch) • Loops (for, while, forEach) • Functions (declaration, parameters, return) • Objects (properties, methods) • Arrays (push, pop, forEach, map) • Classes (basic syntax, constructor) • **Ref:** [MDN JavaScript First Steps](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/First_steps) • [MDN Functions](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Building_blocks/Functions) | **Build:** Console-based Shopping Cart - functions to add items, calculate total, apply discount, display receipt. Use loops, conditionals, and array methods. |
| **W3** | **S6** | JavaScript | **DOM & Event Handling** • What is the DOM? • Selecting elements (getElementById, querySelector, querySelectorAll) • Manipulating content (textContent, innerHTML, innerText) • Manipulating styles and classes (classList) • Creating and removing elements (createElement, appendChild, removeChild) • Event listeners (click, input, submit, keyup) • Event delegation • **Ref:** [MDN DOM Introduction](https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model/Introduction) • [MDN Events](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Building_blocks/Events) | **Build:** Interactive To-Do List - add tasks, mark complete, delete tasks, filter by status. Store tasks in array and display dynamically. | **Objective:** Build a "Dynamic Shopping List" with add, delete, and edit functionality. Include quantity selector and total price calculation. Persist list in localStorage. |
