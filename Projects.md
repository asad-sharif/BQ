## Understanding "Practical Project Outcome" vs "Practice Tasks"

| Column | What It Is | When It Happens | Purpose |
|--------|------------|-----------------|---------|
| **Practical Project Outcome** | The main thing built DURING class (2-hour session) | In-class, guided | Students leave with something working that day |
| **Practice Tasks** | Small assignments done AFTER class | Homework (1-2 hours/week) | Reinforce concepts, prepare for next class |

---

## How Portfolio Projects Fit In

The **4 Portfolio Projects** are built incrementally ACROSS multiple sessions, not in one day.

### Portfolio Project #1: Personal Profile Page

| Session | In-Class Project Outcome | Homework Practice Tasks | Portfolio Progress |
|---------|--------------------------|-------------------------|---------------------|
| **S1** | Build HTML structure of "My Story" page | Write page about favorite hobby | ✅ Started |
| **S2** | Style the page with colors/fonts | Find color scheme, experiment | ✅ Styled |
| **S3** | Make it responsive with Flexbox | Test on phone, fix layouts | ✅ Responsive |
| **S4** | Deploy to GitHub Pages | Send URL to 3 people | ✅ **COMPLETE** |

## Simple Rule

- **In-class projects** = small wins every session
- **Portfolio projects** = 4 big wins across the course
- **Practice tasks** = tiny steps between classes

Students never build a portfolio project from scratch in one day. They build pieces, then assemble.
